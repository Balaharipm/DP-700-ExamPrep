Microsoft Fabric Architecture

December 19, 2024

Microsoft Fabric is a unified analytics platform that brings together Power BI, Azure Synapse Analytics, and Azure Data Factory into a single SaaS offering.

  • OneLake - unified data lake for the entire organization
  • Workspaces - collaborative environments for teams
  • Data Factory - data movement and transformation
  • Synapse Analytics - data warehousing and analytics
