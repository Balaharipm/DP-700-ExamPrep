Fabric Components Quick Reference

August 11, 2025
Component What It Does When to Use in Exam Scenario
OneLake Single, unified data lake for all Fabric data Any data at rest in Fabric lives here
Lakehouse Combines data lake + warehouse in one Store both raw and structured data together
Data Warehouse Pure structured, SQL-based store in Fabric Analytical queries on relational/tabular data
Data Factory ETL/ELT pipelines for batch/scheduled data movement Move data from source to Fabric in bulk
Eventstream Real-time streaming ingestion IoT, clickstreams, live data feeds
Synapse Data Warehouse Query structured data at scale Large datasets needing SQL analytics
Synapse Data Science Notebooks + ML in Fabric Machine learning or big data exploration
Power BI Reports & dashboards Visualizing and sharing insights
Data Activator Data monitoring + triggers Automating alerts/actions when data changes
Notebooks Code-based data analysis Advanced analytics with Python, R, or Spark
